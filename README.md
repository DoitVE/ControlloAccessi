# Hub Gestione Accessi - DOIT VE

Applicazione per la gestione, sincronizzazione e reportistica dei sistemi di accesso EL.MO. e procedure correlate.

## Esecuzione Locale

**Prerequisiti:** Node.js

1. Installare le dipendenze:
   `npm install`
2. Avviare l'app:
   `npm run dev`
